Student Performance
Analysis Project

Analyzing student performance using Python, Excel, and Power BI to uncover key academic insights
and support data-driven educational decisions.

  • View Dataset

Data Collection

Collected a dataset containing student marks, along with additional data points such as preparation level, group, and gender.

  • Tools Used: Excel

Data Processing

Cleaned the dataset by handling missing values, correcting data errors, and transforming variables into appropriate formats (e.g., encoding categorical data).

  • Tools Used: Excel, Python

Insights Generation


Analyzed the data to identify key factors influencing student performance, such as the relationship between attendance and grades, or study hours and academic achievement.

  • Tools Used: Excel, Python

Power BI Dashboard

Developed an interactive Power BI dashboard to visualize key metrics such as overall student performance, grade distributions, and factor-based breakdowns (e.g., by gender, attendance, or study hours).

  • Tools Used: Power BI

Webpage Development

Created a webpage to present the project, including an overview of the analysis, key findings, and the Power BI dashboard. The webpage serves as the final presentation of the entire project.

  • Tools Used: HTML, CSS, JS

Powerpoint Presentation

Created a professional PowerPoint presentation to summarize the project and its findings. The presentation is designed to effectively communicate the projects objectives, methodology and insights.

  • Tools Used: Powerpoint
